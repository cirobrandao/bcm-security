# Security Policy

## Supported Versions

This project is maintained for the BCM Network stack. Security fixes are applied to the latest release on the default branch.

## Reporting a Vulnerability

Please report security issues **privately**.

- Email: security@bcmnetwork.com.br *(or contact via https://cirobrandao.com.br)*
- Include: affected project/version, reproduction steps, impact, and any logs/screenshots.

We will acknowledge receipt, investigate, and publish a fix as soon as possible.
