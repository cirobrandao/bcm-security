# BCM Security

WordPress security/hardening plugin scaffold for **BCM Network**.

## What it does

- Login protection (rate limiting + temporary IP block)
- REST / XML-RPC hardening toggles
- File integrity baseline + scanning
- Alerts (email/log) and an admin UI

## Install

1. Copy the plugin folder to: `wp-content/plugins/securitywp/`
2. Activate the plugin in WordPress
3. Go to: **Settings → BCM Security**
